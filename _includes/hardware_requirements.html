Hardware Requirements

  • Raspberry PI 3B - single board computer to run the mathematical calculations (INR 2750)
  • Arduino Mega - microcontroller board to interface the sensors with the single board computer (INR 750)
  • Ultrasonic Sensors ( x 8) - measure distances to obstacles (INR 300 per sensor)
  • NFC Module - 13.56MHz near field communication with other machines like lifts, lockers etc. (INR 2690)
  • PiCamera - camera module for the Raspberry PI. Used for face identification, feature recognition, path planning etc. (INR 1300)
  • Stepper ( x 5) and Servo Motors ( x 4) - to control orientation of camera and arms (Servo Motors - INR 450 each, Stepper Motors - INR 250 each)
  • High Torque Motors ( x 2) - to run the delivery bot (INR 500 per motor)
  • Motor Driver IC L298N - to drive the motors as desired (INR 300)
  • Battery - to power the robot (INR 450 - Lead Acid, INR 1750 - LiPo)
  • ABS Plastic - to build the body and arms of the bot

Total cost estimate: INR 15,000 - INR 16,000
