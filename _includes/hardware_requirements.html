Hardware Requirements

  • Raspberry PI 3B - single board computer to run the mathematical calculations
  • Arduino Mega - microcontroller board to interface the sensors with the single board computer
  • Ultrasonic Sensors ( x 8) - measure distances to obstacles
  • PiCamera - camera module for the Raspberry PI. Used for face identification, feature recognition, path planning etc.
  • Stepper ( x 5) and Servo Motors ( x 4) - to control orientation of camera and arms
  • High Torque Motors ( x 2) - to run the delivery bot
  • Motor Driver IC L298N - to drive the motors as desired
  • ABS Plastic - to build the body and arms of the bot
